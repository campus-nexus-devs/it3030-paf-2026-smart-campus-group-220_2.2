# Campus Nexus 🎓
### Smart Solutions for Modern Campus Management

**Campus Nexus** is a comprehensive Smart Campus Operations Hub designed to streamline university facility management. This system centralizes facility bookings and maintenance incident ticketing, providing a seamless digital experience for both students and university staff.



## 🚀 Project Overview
Developed as part of the **IT3030: Parallel and Distributed Computing (PAF)** assignment for the academic year 2026.

* **Primary Objective:** To optimize campus facility utilization and simplify maintenance reporting through a centralized hub.
* **Key Modules:**
    * **Facility Booking System:** Real-time scheduling for university halls, labs, and equipment.
    * **Maintenance Incident Ticketing:** A streamlined workflow for reporting and tracking facility repairs.



## 🛠️ Technical Stack
* **Backend:** Spring Boot (Java) - RESTful API Architecture
* **Frontend:** React.js - Modern Web Client Application
* **Database:** SQL Based Relational Database (MySQL/PostgreSQL)
* **Build Tool:** Maven



## 📂 System Architecture
The project follows a **Client-Server Architecture**:
1.  **Campus Nexus API:** The backend engine handling business logic, security, and database interactions.
2.  **Campus Nexus Client:** The interactive user interface for students, faculty, and maintenance teams.
